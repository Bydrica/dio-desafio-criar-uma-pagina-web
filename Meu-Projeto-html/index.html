Ferramentas Utilizadas

Para começar a ciar páginas web em HTML, não é preciso utilizar ferramentas complexas e caras. Um editor de texto simples e um navegador são o suficiente para criar páginas HTML.

No entanto, utilizar um editor de código que ofereça um conjunto robusto de funcionalidades pode ajudar a melhorar a qualidade do código.

Um ótimo exemplo é o VS Code, ele é uma ferramenta de edição de código gratuita, e muito popular devido sua extensibilidade e rica funcionalidade.

Inspetor de Elementos

O inspetor de elementos é uma ferramenta integrada nos navegadores que permite examinar, editar e depurar uma página web em tempo real.

Essa ferramenta é muito útil para desenvolvedores iniciantes, pois ajuda a entender o que está acontecendo na página web, e também para os mais experientes, permitindo que problemas sejam identificados e resolvidos rapidamente.

Estrutura Básica do HTML

A estrutura básica de uma página HTML consiste em uma série de elementos que definem o conteúdo e a organização da página web.

Alguns exemplos de tags que compõem a estrutura básica do HTML são:

  • <html> O Elemento raiz do documento.
  • <head> O cabeçalho que contém metadados, links CSS e outras informações.
  • <title> Elemento que define o título da página que aparece na aba do navegador.
  • <body> Elemento que contém todo o conteúdo visível, como imagens, textos, vídeos e etc.

Falando sobre Tags

As tags (ou elementos) são a base do HTML, cada uma possui uma função específica, e elas são utilizadas para definir a estrutura e o conteúdo de uma página web.

Além das tags estruturais citadas anteriormente, existem outros elementos essenciais na criação de páginas, algumas delas são:

Tags de Texto:

  • <h1> até <h6>: Define títulos e subtítulos, e seguem uma hierarquia.
  • <p>: Define um parágrafo de texto.

Tags de Listas:

  • <ul>: Define uma lista não ordenada (geralmente pontos).
  • <ol>: Define uma lista ordenada (geralmente números).
  • <li>: Define um item de lista, usado dentro de <ul> ou <ol> .

Tags de Links e Imagens :

  • <a>: Define um hyperlink.
  • <img>: Define uma imagem.

Atributos das Tags

Os atributos em HTML são usados para fornecer informações adicionais sobre as tags. Eles aparecem dentro das tags de abertura e consistem em um nome e um valor. Além disso, são essenciais para definir propriedades e comportamento de elementos em uma página web.

Alguns exemplos de atributos comuns e específicos são:

  • id: Identificador único para um elemento.
  • class: Define uma classe para um elemento.
  • style: Define estilos CSS diretamente no elemento.
  • title: Define um texto de dica que aparece quando o usuário passa o mouse sobre o elemento.
  • href: Especifica o destino de um link. Usado dentro da tag <a>.
  • src: Define a fonte de um recurso externo, como uma imagem. Usado dentro da tag <img>.
  • alt: Fornece um texto alternativo para uma imagem, usado quando a imagem não pode ser exibida.
  • width e height: Define a largura e a altura de elementos, como imagens ou vídeos.

Textos

Os elementos de texto são fundamentais para definir, estruturar e estilizar o conteúdo textual das páginas web. Eles oferecem várias opções para formatação, citação e referências.

Alguns dos elementos de texto mais comuns são:

  • <h1> até <h6>: Definem títulos e subtítulos.
  • <p>: Define um parágrafo de texto.
  • <strong>: Define texto em negrito.
  • <i>: Define texto em itálico.
  • <u>: Define texto sublinhado.
  • <mark>: Define texto destacado com fundo amarelo.
  • <small>: Define texto menor.
  • <sup>: Define texto sobrescrito.
  • <blockquote>: Define uma seção de texto citada de outra fonte.
  • <sup>: Define texto sobrescrito, geralmente usado em expoentes. Exemplo: E=mc2.
  • <sub>: Define texto subscrito, geralmente usado em fórmulas químicas ou matemáticas. Exemplo: H2O.

Listas Ordenadas e não Ordenadas

As Listas Ordenadas e não Ordenadas são muito úteis para organizar e apresentar o conteúdo de maneira clara e estruturada.

Sobre as listas não Ordenadas:

As listas não ordenadas são usadas para agrupar itens que não possuem uma ordem específica. Cada item na lista é marcado com um marcador, geralmente representados por pontos.

Sobre as Listas Ordenadas:

As listas ordenadas são usadas para agrupar itens que possuem uma ordem específica. Cada item é numerado, representando uma classificação.

Links

Os links, representados pela tag <a>, são essenciais para a navegação na web, permitindo criar hyperlinks para outras página, recursos ou locais na internet.

Exemplo de Link

<Visite DIO.me>

Conceitos Importantes

  1. O texto exibido entre as tags <a> é o que o usuário vê, por isso, é muito importante dar nomes significativos que descrevam para onde o link leva.
  2. href é o atributo que define o destino do link.
